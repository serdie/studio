# **App Name**: AI Learning Hub

## Core Features:

- User Registration and Authentication: Secure registration and login for students with user profile management.
- Course Module Access: Structured access to the three modules: 'Introducción a la IA,' 'Modelos de Lenguaje LLMs,' and 'Creación de avatares virtuales'.
- Progress Tracking: Monitor individual student progress within each module.
- Resource Library: Access to downloadable resources, example files, and datasets for each module.
- AI Learning Assistant: AI tool that provides summaries, answers questions, and generates relevant images based on the course content for each section.
- Data Upload/Download: Functionality to upload data for exercises and download completed assignments and reports.
- Generative AI Avatar Creation Tool: Students will use generative AI tools to create and customize virtual avatars, as outlined in Módulo 3.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), embodying intelligence and creativity.
- Background color: Light gray (#ECEFF1), providing a clean, modern learning environment.
- Accent color: Electric blue (#2196F3), used for interactive elements and highlights, calling attention and creating contrast.
- Headline font: 'Space Grotesk' sans-serif font to maintain a computerized and scientific aesthetic.
- Body font: 'Inter' sans-serif to enhance the readability and user experience.
- Use a set of modern and minimalist icons to represent different actions and content types.
- Employ a clean and intuitive layout, optimized for learning with clear visual hierarchy.
